# Documentación del Proyecto - Demo Mexico Limited

## Arquitectura
Este proyecto es una demo de la plataforma CRM de Mexico Limited.
Se divide en dos partes principales:
1. **Frontend**: Aplicación web desarrollada con Next.js (React) y CSS Vanilla enfocada en un diseño "premium" con glassmorphism.
2. **Backend**: API en Flask (Python) que expone un modelo de similitud de coseno para empatar las necesidades del emprendedor (recopiladas a través del diagnóstico) con los servicios ofrecidos.

Ambos se integran con **Supabase** para la autenticación y la base de datos (mockeados temporalmente para fines de la demo si no hay llaves provistas).

## Estructura de Directorios
- `/frontend`: Código fuente de Next.js.
- `/backend`: API de Flask y scripts de Python.
